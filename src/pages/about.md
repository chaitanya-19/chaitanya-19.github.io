I'm Chaitanya Kulkarni, an AI Engineer based in SF Bay area, CA. I build production agentic pipelines and RAG systems on AWS — OpenSearch, Bedrock, Lambda — using LangGraph, MCP, FastAPI, and Kubernetes.

This blog is where I work through modern LLM architecture from first principles. Counting parameters, deriving memory math, walking through the linear algebra. Each post takes one piece of vanilla GPT-2 and shows what changes in a frontier-scale model — and why.

The series is targeted at engineers who use LLMs and want to understand what's actually inside them. If you've trained or fine-tuned models, you can skim the introductions. If you've only ever called an API, the posts assume you've at least read [Attention Is All You Need](https://arxiv.org/abs/1706.03762) — beyond that, every shape and every count is derived from scratch.
